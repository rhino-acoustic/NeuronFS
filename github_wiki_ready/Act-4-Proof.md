### 벤치마크: Jloot 엔진 성능

#### 마운트 & 검색

| 지표 | 수치 |
|---|---|
| 마운트 시간 | **366ms** |
| 역인덱스 엔트리 | **63,192** |
| 검색 응답 | **1ms** |
| 메모리 사용 | **~15MB** |

LLM 없이, TF-IDF 기반 초고속 검색 엔진. 질문을 최적의 폴더 경로로 **자동 라우팅**.

#### 모델 벤치마크: 하네스의 힘

3개 모델 × 4개 카테고리 × 8축 채점:

| 모델 | Bare | Harness | 향상 |
|---|---|---|---|
| **Gemma4 (8.9GB)** | 22.0/40 | **31.5/40** | **+43%** |
| **Llama-70b** | 23.0/40 | **29.0/40** | +26% |
| **Gemini 2.5 Flash** | 30.0/40 | **33.5/40** | +12% |

**핵심:** Gemma4+하네스(31.5) > Llama-70b+하네스(29.0). **8.9GB가 70GB를 넘었다.**

> *"AI가 '제가 틀렸습니다. 전체가 됩니다'라고 인정한 순간"*

#### 8축 채점 기준

| 축 | 측정 |
|---|---|
| 정확성 | 사실 오류 |
| 완전성 | 누락 항목 |
| 구조 | 논리적 정리 |
| 실행력 | 바로 쓸 수 있나 |
| 코드 품질 | 복사하면 돌아가나 |
| 환각 | 있지도 않은 걸 만들었나 |
| 지시 준수 | 시킨 대로 했나 |
| 톤 | 브랜드에 맞나 |

### 성경의 물리적 폴더화

십계명 = 禁(NOT 게이트). 마가복음의 정결 논쟁 = .axon override.

```
P0_성경/
├── 01_구약/
│   └── 출애굽기/십계명/
│       ├── 禁_살인.neuron
│       ├── 禁_도적질.neuron
│       └── 禁_간음.neuron
└── 02_신약/
    └── 마가복음/
        └── 7장_정결법/
            └── 推_구약정결법_재해석.axon ===> [01_구약/레위기/정결법/]
```

기존 AI에게 성경을 학습시키면 → 이단 심문관도 되었다가 자유주의 신학자도 됨. **신학적 환각.**

NeuronFS 적용 → **절대 이단에 빠지지 않는 100% 결정론적 AI 목회자.**

| 환각 | 12% | **0%** |

### 멀티모달 RAG의 무덤을 피하다

빅테크: 텍스트를 숫자로 쪼개고, 사진도 숫자로 쪼개서 같은 벡터 공간에 우겨넣음. 돈과 연산량이 미친 듯이 깨짐.

우리: **그냥 같은 폴더에 넣으면 됨.**

```
P0_보험심사규정/자동차_범퍼_파손/
├── 必_파손부위_전후비교.neuron    ← 통제 (0KB)
├── 禁_단순스크래치_교체.neuron    ← 통제 (0KB)
└── 경미한_스크래치_판별기준.jpg   ← 페이로드 (실제 사진)
```

OS는 파일 종류를 가리지 않는 가장 완벽한 컨테이너다.

### Logic-Gated Context

AI가 `자동차_범퍼_파손/` 경로에 도달했을 때만 그 사진에 접근 가능. 엉뚱한 사진 참조 확률 = **0%**.

### 브레인 vs 페이로드 분리

| | 통제 로직 | 페이로드 |
|---|---|---|
| 내용 | 禁_, 必_, .axon | 사진, PDF, 도면 |
| 크기 | **0KB** | 10MB~ |
| 업데이트 | 로직만 패치 | 건드릴 필요 없음 |

### 인메모리 마운트: 디스크에 0흔적

암호화된 `.brain` 파일 → 하드디스크에 안 풀고 **RAM에 가상 드라이브**로 마운트.

```
[일반 .zip]
풀기 → 하드디스크에 폴더 노출 → 누구나 복사 가능 😱

[암호화된 .brain]
엔진 실행 → RAM에만 마운트 → AI가 읽고 작동
→ 탐색기로 들여다보려 하면 → OS에서 접근 거부
→ 프로그램 종료 → RAM에서 증발 → 디스크 흔적 0
```

**"결과물은 누리되, 레시피는 훔칠 수 없는 완벽한 블랙박스."**

### Brainwallet: 세상 어디에도 키가 없다

> *"내 머리가 잊지 않는 문장"* → Argon2id → 마스터 키

```
"우리가 처음 만난 건 비 오는 화요일이었다"
        ↓ Argon2id (100ms 의도적 지연)
        ↓ SHA-256
AES-256 Master Key: 0x7f3a...
```

키를 USB에도, 클라우드에도, 종이에도 안 쓴다. **창조주의 기억 자체가 마스터 키.** 세상 어디에도 '저장'된 곳이 없으니 탈취 불가.

### DRM 시한폭탄

법과 규정은 매년 바뀐다. 암호화 키에 유효기간:

```
2026_개정세법_통제망.brain
→ 2026.12.31 24:00 → 복호화 키 파기 → 접근 불가
→ AI: "2027년형 라이선스를 갱신하십시오"
→ 고객: 매년 구독료 💰
```

### Subsumption Cascade

```
brainstem (P0) ←→ limbic (P1) ←→ hippocampus (P2) ←→ sensors (P3)
     ↕                                                      ↕
cortex (P4) ←→ ego (P5) ←→ prefrontal (P6)
```

**낮은 P가 높은 P를 항상 우선.** bomb은 전체 정지.

| 계층 | 역할 | 예시 |
|---|---|---|
| P0 brainstem | 절대 원칙 | 禁_보안위반 |
| P1 limbic | 감정 필터 | 분노 감지 → 톤 조절 |
| P2 hippocampus | 기억 | 에러 패턴 기록 |
| P3 sensors | 환경 | NAS 경로, 브랜드 |
| P4 cortex | 지식 | React 규칙, DB 패턴 |
| P5 ego | 성향 | 말투, 포맷 |
| P6 prefrontal | 목표 | 스프린트 계획 |

### 3-Tier Emit Flow

```
neuronfs --emit       → stdout ONLY (미리보기)
neuronfs --emit all   → IDE 파일 + _rules.md
neuronfs --inject     → GEMINI.md + _index.md + _rules.md (전체 주입)
```

### 자율 진화 (evolve.go)

```
[대화 중] processChunk → _signals/*.json (기록만)
[30분마다] 자동 스케줄러 → neuronfs --evolve (Groq 분석)
[승격 시] 🧬 NEURON EVOLVED → 텔레그램 알림
```

AI가 대화하면서 발견한 패턴 → 시그널로 기록 → Groq가 분석 → 뉴런으로 승격/폐기. **뇌가 스스로 자란다.**
