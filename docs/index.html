Workflow

Three commands. Zero infrastructure.

01

Create

Define your AI's governance as folders. Each folder name is a rule. Subfolder depth = specificity.

mkdir brain/brainstem/禁/data_leak
mkdir brain/cortex/dev/推/testing
mkdir brain/limbic/必/korean_response
02

Compile

NeuronFS walks the tree, resolves subsumption priorities, and emits a format-specific output.

neuronfs --emit gemini
✓ 420 neurons → GEMINI.md (12KB)
✓ 3-tier rules: ALWAYS/WHEN/NEVER
03

Deploy

The emitted file is injected into your AI tool. The agent reads it. Governance active. Zero tokens consumed.
