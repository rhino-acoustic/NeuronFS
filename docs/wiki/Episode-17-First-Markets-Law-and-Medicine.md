### 왜 법과 의료인가?

법과 의료는 AI 도입에 가장 보수적인 산업이다. 이유는 단 하나: **"AI가 틀리면 사람이 죽거나 감옥에 간다."**

```
[기존 AI 챗봇의 법률 답변]
질문: "다주택자 양도세 비과세 요건은?"

ChatGPT: "1세대 1주택으로 2년 보유 시 비과세입니다."
-> 조정대상지역 여부 미확인
-> 거주요건 2년 누락
-> 고객이 이걸 믿고 집을 팔면 -> 세금 폭탄 -> 소송

변호사: "AI 답변 믿고 자문했다고요? 저는 책임 못 집니다."
```

**이것이 법조계가 AI를 거부하는 이유다.** 확률적 답변으로는 6,000만 원짜리 세금 실수를 막을 수 없다.

### 트로이 목마 전략: 3단계 시장 침투

**1단계: 무료 미끼 (양도소득세 오픈소스)**

```
P0_조세법/양도소득세/
+-- 조정대상지역/
|   +-- 禁/거주2년미만_비과세적용.neuron
|   +-- 必/조정지역_확인.neuron
|   \-- _context_2026개정.neuron
+-- 비조정대상지역/
|   \-- 必/보유2년_확인.neuron
\-- 다주택자/
    +-- 禁/3주택이상_비과세.neuron
    \-- 推/장기보유특별공제.axon ==> [보유기간/]
```

-> GitHub에 공개 -> 세무사들이 "와, 이거 진짜네" -> 바이럴

**2단계: 놀라움 확산**

-> 세무사 A: "프롬프트 엔지니어링이 필요 없다고?"
-> 세무사 B: "폴더 구조만으로 AI가 세법을 100% 정확히 답한다고?"
-> 블로그/유튜브: "mkdir이 벡터 DB를 이겼다" -> 화제

**3단계: 유료 전환 (B2B)**

대형 로펌 연락: "가업상속공제 + 국제조세도 만들어 주세요."
-> 이건 오픈소스가 아닙니다.
-> .jloot 카트리지로 암호화 납품
-> 연간 구독: $999/년 (법 개정 시 자동 업데이트)

> *"기술은 무료로 풀어서 종교를 만들고, 복잡한 도메인 지식의 회로도를 B2B로 판다."*

### .axon 크로스 도메인: 세법과 임대법의 충돌

실전에서 법률은 절대 혼자 작동하지 않는다. **다주택자의 양도세와 임대차보호법이 동시에 적용되는 순간**:

```
다주택자가 임대 중인 집을 팔려 한다.

[세법 마더보드]
P0_조세법/양도소득세/다주택자/
\-- 推/장기보유특별공제.axon ==> [보유기간/]

     | .axon 텔레포트 |

[임대법 마더보드]
P0_임대차보호법/임대료/
\-- 禁/5프로초과_인상.neuron   <-- NOT 게이트

     | 충돌 감지 |

결과: 임대료 5% 초과 인상 이력 발견
-> 장기보유특별공제 .axon이 禁 게이트에 부딪힘
-> 비과세 차단
-> AI 답변: "임대료 인상 이력으로 인해 특별공제 적용 불가합니다."
```

**RAG는 이 두 법률의 교차 적용을 꿈도 꾸지 못한다.** 벡터 DB는 "유사한 조문"을 찾을 뿐이지, 두 법률 체계의 물리적 충돌을 감지할 메커니즘이 없다.
