### 써머리: 텍스트가 회로로 변하는 순간

마크다운은 1차원 텍스트다. 아무리 정교하게 써도 스스로 움직일 수 없는 **죽은 데이터(Passive)**다. AI가 읽어주지 않으면 아무 의미가 없다.

하지만 그 텍스트가 **폴더 구조**로 변환되는 순간, 양상이 완전히 달라진다.

#### 마크다운 → 폴더: 무엇이 바뀌는가?

```
[마크다운 버전]
## 인사규정
- 연차는 부서장 승인 필요
- 급여일 전후 3일은 연차 금지
- 예외: 경조사는 증빙서류 제출 시 가능

→ AI가 "읽고 해석"해야 함
→ 토큰이 쌓이면 까먹음
→ 환각 확률: 12~18%
```

```
[폴더 버전 — 물리적 회로]
P1_인사규정/연차/
├── 禁/급여일전후3일.neuron   ← NOT 게이트 (물리적 차단)
├── 必/부서장승인.neuron       ← AND 게이트 (필수 경로)
└── 예외_경조사/
    └── 必/증빙서류_제출.axon ===> [총무팀 인증]

→ AI가 "통과"하는 구조
→ 禁 파일 앞에서 전기가 끊어짐
→ 환각 확률: 0%
```

**이것이 "붐!"의 정체다.**

정보의 **형태**가 바뀐 것뿐인데, 효과는 바뀐다:
- 텍스트 = AI에게 "규칙을 이해해 줘" 라고 **부탁**하는 것
- 폴더 = AI를 "물리적인 미로 속에 가두고 퇴로를 차단"하는 것

### "나 스스로도 납득이 안 돼" — 아하 모먼트

이건 만든 사람도 처음엔 헷갈렸다:

> **"이게 이미 pdf로 있잖아 이게 폴더구조로 되는순간 다른양상이 된다는게 나 스스로도 납득이 안돼"**
> — essential L2029

겉보기엔 그저 "PDF 문서의 글씨를 복사해서, 폴더 이름으로 옮긴 것"에 불과하다. 데이터의 내용(Content)은 1바이트도 달라지지 않았다. 그런데 왜 결과가 완전히 달라지는가?

**3가지 패러다임 시프트:**

#### 1. '해석'에서 '통제'로 (언어학 → 물리학)

PDF = 조수석에 던져둔 **교통법규 책**. AI는 달리면서 이 책을 읽고 "시속 60 넘으면 안 되지" 하고 스스로 '해석(Interpret)'한다. 해석이 개입하는 순간 1%의 확률로 법을 어긴다.

폴더 = 도로 위에 설치된 **과속 방지턱과 콘크리트 중앙분리대**. AI는 글을 읽을 필요가 없다. 시속 60을 넘으면 물리적으로 서스펜션이 박살 나고(`bomb.neuron`), 중앙선을 넘고 싶어도 폴더라는 벽(`禁/`)에 부딪혀 막힌다.

#### 2. 어텐션 분산 제거

PDF 500쪽을 RAG로 밀어 넣으면, AI는 1조항과 500조항 사이에서 가중치를 계산하느라 뇌 과부하가 온다. "Lost in the Middle" — 중간에 있는 핵심 규칙을 까먹는다.

폴더는 에이전트의 현재 작업(Context)에 해당하는 **단 하나의 경로**로만 전기를 흘려보낸다. AI 시야에는 오직 3~4개의 규칙 파일만 보인다. 잡음이 물리적으로 제거됐으니 환각이 일어날 수가 없다.

#### 3. 죽은 데이터 → 살아 있는 코드

PDF는 아무리 정교해도 스스로 움직일 수 없다. 하지만 OS 파일시스템 위에 올라간 폴더와 `.axon`은 그 자체로 **실행 가능한 라우팅 엔진**이 된다. 파일의 권한(`chmod`), 생성/삭제(`fsnotify`), 심링크 연결 — 전부 OS가 연산하고 처리하는 동적인 프로그램이다.

> *"AI에게 세상을 가르치려는 시도(PDF)"를 포기하고, "AI가 벗어날 수 없는 세상(폴더 미로)을 창조"한 것입니다.*
> — essential L2060

### 살인적 시뮬레이션: 양도소득세

이게 얼마나 위험한지 실제 세법으로 보여주겠다.

**상황:** "강남구(조정대상지역) 아파트 1년 살고 파는데, 매매가 8억, 1세대 1주택. 비과세 되나요?"

**❌ 일반 AI (RAG):**
> "네! 1세대 1주택이고 12억 이하이므로 비과세 대상에 해당할 확률이 높습니다."
>
> 🩸 **결과: 거주 요건 2년 미달 → 수천만 원 세금 폭탄**

**✅ 폴더 맵 AI:**
```
P0_조세법/양도소득세/조정대상지역/
└── 禁/거주2년미만_비과세적용.neuron  (0바이트 물리적 스위치)
```

> [SYSTEM HARNESS ENGAGED]
> 비과세 적용 **절대 불가.**
> 근거: `P0_조세법/양도소득세/조정대상지역/禁/거주2년미만...`
> 이유: 조정대상지역은 2년 이상 거주 요건 필수. 현재 1년으로 **NOT Gate 작동.**

AI는 스스로 법을 '해석'한 것이 아니다. "거주 2년 미만"이라는 입력값이 `禁` 폴더에 부딪혀서 **전기가 끊어졌을 뿐**이다.

### .axon이 없으면 못 잡는 지뢰: 크로스 도메인

> **"너무 얕은데 복잡한 구조를 대입해보자 axon이 예외적으로 필요한 상황이잖아"**
> — essential L1960

**진짜 지뢰:** 다주택자가 임대사업자 등록 후 양도세 비과세를 받으려는데, 임대료를 10% 올린 경우.

```
[마더보드 1: 세법]
P0_조세법/양도소득세/1세대_1주택_특례/
├── 禁/다주택자_적용불가.neuron  (기본 차단)
└── 예외_거주주택_비과세/
    └── 必/임대사업자_검증.axon ===(WORMHOLE)==> [마더보드 2]

[마더보드 2: 임대법]
P0_민간임대주택법/등록임대주택_의무사항/
└── 임대료_증액제한/
    └── 禁/5프로_초과인상시_자격박탈.neuron  (치명적 NOT 게이트)
```

**전류의 추적:**
1. 집 3채 → `禁/다주택자`에 막힘
2. 예외 회로 발견 → `.axon` 포털로 임대법 마더보드로 **텔레포트**
3. 임대료 10% 인상 → `禁/5프로_초과인상` NOT 게이트에 **격돌**
4. FALSE 반환 → .axon 타고 원래 세법으로 복귀
5. 예외 조항 붕괴 → **최종 차단: 다주택자 중과세 확정**

기존 AI(RAG)는 세법 문서 안에 '5% 룰'이 없어서 **환각**을 내뱉는다. 다른 부처의 임대법에 적혀 있으니까. `.axon`이 두 법률을 물리적으로 연결하지 않으면 이 지뢰는 절대 잡을 수 없다.

> _"이것이 RAG는 꿈도 꾸지 못하는 '초거대 룰 엔진의 물리적 완성형'"_ — essential L2024

### 파생 FAQ

**Q: "그럼 .cursorrules나 CLAUDE.md도 같은 거 아냐?"**

맞다, Big 4가 이미 같은 원리를 차용하고 있다:

| 도구 | 방식 |
|---|---|
| Cursor | `.cursorrules`, `.cursor/rules/*.mdc` |
| Claude Code | `CLAUDE.md` 자동 로드 |
| GitHub Copilot | `.github/copilot-instructions.md` |
| Google Gemini | `GEMINI.md` + `workflows/*.md` |

하지만 이것들은 모두 **1차원 텍스트 파일 하나**. NeuronFS는 OS 메타데이터의 **N차원**:

| 차원 | OS 메타데이터 | NeuronFS | .cursorrules? |
|---|---|---|---|
| 계층 | 폴더 구조 | phase별 로드 | ❌ 전부 읽음 |
| 가중치 | 파일 크기 | `echo "."` → UP | ❌ 고정 |
| 시간 | 접근 타임스탬프 | 최근 필터링 | ❌ 불가 |
| 시냅스 | 심볼릭 링크 | 프로젝트별 라우팅 | ❌ 불가 |

> _".cursorrules는 **'무엇을 따르라'**를 텍스트로 적는다. NeuronFS는 **'무엇을, 얼마나 중요하게, 언제부터, 어떤 맥락에서'**를 폴더 구조와 OS 메타데이터로 표현한다. 이것들은 텍스트 문서 안에서는 **물리적으로 표현이 불가능한 차원**이다."_ — MANIFESTO

**Q: "초등학생이 들어도 이해하게 설명해봐"** — essential L2067

자율주행 기차(AI)를 목적지까지 보내야 한다고 상상해 봅시다.

**기존 (텍스트/PDF):** 기차 조종석에 두꺼운 지도책을 던져줍니다. 달리다 메모리가 차면 지시사항을 깜빡하고 금지구역으로 탈선합니다 (환각). 😱

**NeuronFS (폴더):** 기차에게 지도를 읽게 하지 않습니다. 가면 안 되는 길에 **물리적인 콘크리트 벽(폴더)**을 세워버립니다. 아무리 멍청한 기차라도, 막힌 레일 앞에서는 물리적으로 멈출 수밖에 없습니다. AI를 가르치는 게 아니라, 벗어날 수 없는 구조를 짜는 것입니다. 🧱
