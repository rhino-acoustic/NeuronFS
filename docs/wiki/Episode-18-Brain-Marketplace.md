### "엔진은 무료로, 뇌(경험)를 판다"

**워드프레스 비유:**
- 워드프레스 자체는 **무료**. 하지만 잘 만든 유료 테마와 플러그인 시장이 수조 원대.
- NeuronFS 엔진은 AGPL로 **무료**. 하지만 전문가의 뇌(.jloot 카트리지) 시장이 수조 원대.

```
[무료] NeuronFS 엔진 (4MB Go 바이너리, AGPL-3.0)
  -> 생태계 장악 -> "폴더 = 뉴런"이라는 패러다임을 업계 표준으로 각인

[유료] 마스터 브레인 (.jloot 카트리지)
  -> "Next.js + Supabase 환각 방지 뇌" (49USD)
  -> "금융권 보안 코딩 가이드 뇌" (99USD)
  -> "중대재해처벌법 방어 시나리오 뇌" (999USD)
```

### 수익 구조

```
전문가 (판매자)
  -> 10년간 축적한 노하우를 .jloot로 패키징
  -> Brain Marketplace에 등록
  -> 판매 수익의 85% 수취

플랫폼 (NeuronFS)
  -> 거래 수수료 15%
  -> 카트리지 암호화/DRM 인프라 제공
  -> 리뷰/별점/인증 시스템 운영

구매자 (기업/개인)
  -> 카트리지 구매 -> 로컬에 마운트 -> AI가 즉시 전문가 수준으로 작동
  -> 데이터는 밖으로 안 나감 (온프레미스)
  -> 구독 갱신 -> 법/규정 변경 시 자동 업데이트
```

### 구독형 DRM -- 왜 매년 갱신해야 하는가

법과 규정은 매년 바뀐다. 의료 가이드라인도 매년 개정된다.

```
2026_개정세법_통제망.jloot
-> 유효기간: 2026.12.31 24:00
-> 만료 시: Argon2id 키 폐기 -> 카트리지 자동 잠금
-> AI 응답: "2027년형 브레인 카트리지를 갱신하십시오."
-> 고객: 갱신 버튼 클릭 -> 최신 법률 뇌 자동 다운로드

핫픽스:
-> 2026년 7월 긴급 세법 개정 발표
-> 전문가가 禁/必 뉴런 3개만 추가
-> 구독자에게 OTA 패치 (0바이트 폴더 3개 추가)
-> 리임베딩 비용: 0원 (벡터 DB였다면 수십만 원)
```

> *"App Store가 앱을 팔듯이, Brain Marketplace는 뇌를 판다. 차이점은 -- 이 뇌는 환각을 하지 않는다."*
