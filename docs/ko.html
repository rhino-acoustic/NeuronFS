읽히는 동시에 실행됩니다

폴더로 AI 행동을 구조화합니다. 각 폴더가 규칙이고, 각 경로가 문장이고, 폴더구조 자체가 문맥입니다. 법률은 읽히지만 실행 불가. 코드는 실행되지만 읽기 어려움. 뇌 폴더는 둘 다 — 버전 관리되고, 상속 가능하고, 벤더에 독립적인 거버넌스.

명제

옳은 질문이 옳은 답보다 더 희귀합니다

1. 가치관의 최초 실행 가능한 포맷

인류는 수천 년간 가치관을 저장했지만 실행할 수 없었습니다 — 법률, 경전, 사규 모두 해석이 필요했습니다. 뇌 폴더는 처음으로 가치관이 해석 없이 직접 실행되는 포맷입니다.

2. 모델 독립적 지식 자산

GPT → Claude → Gemini → 미래 모델. 뇌는 그대로. 조직의 판단 기준이 어떤 AI 벤더와도 독립적으로 누적됩니다. 지식이 모델에 종속되지 않는 최초의 구조.

3. AI 시대의 책임 소재 인프라

"AI가 왜 그 결정을 했는가?" 증명해야 하는 시대가 옵니다. 뇌 폴더 + git 로그는 유일한 감사 가능한 사전 문서화 거버넌스 구조입니다. XAI(모델 해석)가 아닌, 모델에 준 규칙의 문서화.

4. 전문가 지식의 사멸 방지

30년 경력자가 은퇴하면 판단 기준이 사라집니다. 뇌 폴더는 의사결정 구조를 상속 가능하고 실행 가능한 형태로 보존합니다. 뇌의 유언장.

5. 아직 없는 표준

CLAUDE.md, AGENTS.md, .cursorrules — 업계가 파일 기반 AI 거버넌스로 수렴하고 있습니다. 하지만 AI 운영 지식의 표준 포맷은 아직 없습니다. 그 포맷을 정의하는 자가 다음 시대를 만듭니다.

기능

왜 폴더가 벡터를 이길까

📁

폴더 = 규칙

mkdir로 규칙을 만들고, git diff로 거버넌스를 추적합니다. 모든 변경은 감사 가능하고, 사람이 읽을 수 있습니다.

⚡

빠른 조회

개별 규칙은 OS dentry 캐시로 조회. 전체 emit은 O(n)이지만 420 뉴런이 40ms 미만에 완료됩니다.

🔒

망분리 배포

네트워크 호출 제로, 텔레메트리 제로. USB에 담아 배포. 데이터가 외부로 나갈 수 없는 환경에 적합합니다.

🎯

멀티 포맷 출력

하나의 두뇌, 6개 출력: GEMINI.md, .cursorrules, CLAUDE.md, AGENTS.md, COPILOT.md, .windsurfrules.

🧬

자가 진화

헤비안 학습, TTL 감쇠, 폐쇄 학습 루프. 자주 쓰이는 뉴런은 강화되고, 안 쓰이는 뉴런은 사라집니다.

💰

영원히 무료

클라우드 없음, 구독 없음. 4MB Go 바이너리. AGPL-3.0 오픈소스. 벡터 DB 대비 최소 인프라.

벤치마크

실제 테스트 데이터. 실제 숫자.

모든 벤치마크는 내부 테스트(go test ./...)입니다. 독립적 재현을 권장합니다.

활용 사례

책임 소재가 중요한 곳

AI 책임법이 전 세계적으로 강화되고 있습니다. NeuronFS는 git 커밋 로그를 통해 감사 가능한 거버넌스 증거를 제공합니다.

⚖️

법무 & 컴플라이언스

"AI가 그 결정을 내릴 때 어떤 규칙이 활성화되어 있었는가?" → git log brain/. EU AI Act 감사 증거가 내장됩니다.

🏥

의료

추적 가능한 규칙 세트를 가진 AI 진단 보조. "왜 AI가 이것을 플래그했는가?" → 해당 조건의 禁 폴더, 버전 관리됨.

🏦

금융 & 보험

대출 거절 AI, 보험금 거부 AI — 모든 결정이 특정 뇌 폴더 상태로 추적 가능. 컴플라이언스 담당자가 정확한 거버넌스 버전을 감사할 수 있습니다.

🎓

교육

학생이 직접 禁/推 규칙을 설계. AI가 그 규칙으로 동작하면서 논리적 허점을 드러냅니다. 소크라테스식 교육.

🚨

재난 대응

인터넷 다운? USB 뇌 + 로컬 모델 = 사전 정의된 규칙을 가진 완전 작동 AI. 혼란 속에서도 일관된 판단.

🔄

벤더 독립

GPT에서 Claude로, Gemini로 전환해도 뇌는 동일. 조직의 거버넌스 지식이 어떤 AI 벤더와도 독립적으로 누적됩니다.

비전

두뇌가 상품입니다

수천 번의 실패를 거쳐 정제된 폴더 구조는 전달 가능한 전문 지식입니다. 사람이 읽을 수 있고, 기계가 실행할 수 있고, 버전 관리되는 조직 가치관의 최초 표현.

💼

🛒

뇌 마켓플레이스

별점, 리뷰, 업종별 카테고리. AI 거버넌스를 위한 플러그인 마켓. 커뮤니티가 큐레이션한 베스트 프랙티스.

🧬

뇌의 유산

은퇴하는 전문가의 판단을 폴더로 인코딩. 30년의 의료 판단이 보존되고 실행 가능. 사람이 떠나도 지식은 남습니다.
