인간의 가치관이
읽히고 실행되는 최초의 인프라

폴더 구조로 AI의 행동을 제어합니다. 각각의 폴더는 규칙이며, 경로는 문장이고, 트리 구조 자체가 거대한 문맥입니다. 법률은 읽을 수 있으나 스스로 실행되지 않고, 코드는 실행되나 맥락을 읽기 어렵습니다. NeuronFS의 뇌 폴더는 이 둘을 동시에 충족하는, 모델에 종속되지 않는 최초의 거버넌스입니다.

올바른 답보다 올바른 질문이 더 희귀합니다

1. 가치관의 최초 실행 가능한 포맷

인류는 수천 년 동안 가치관을 기록해왔지만 스스로 실행될 수는 없었습니다. 헌법과 사규 모두 인간의 능동적인 해석을 거쳐야만 했습니다. 뇌 폴더는 시대의 가치관이 별도의 해석 레이어 없이 직접 AI에게 실행되는 역사상 첫 번째 포맷입니다.

2. 모델에 종속되지 않는 지식 자산

GPT에서 Claude로, 다시 Gemini로. 모델은 계속 바뀌지만 조직의 뇌는 영구히 남습니다. 고유한 의사결정 기준이 파일 시스템에 선언적으로 누적되며, 어떤 AI 벤더에도 코어 지식을 종속시키지 않습니다.

3. 통제와 책임의 인프라

"AI는 왜 그런 결정을 내렸는가?" 이제는 기업이 이를 증명해야 합니다. 뇌 폴더와 Git 로그는 감사와 추적이 가능한 유일무이한 거버넌스 체계입니다. 사후 결과에 대한 통계적 해석이 아니라, 애초에 어떤 규칙이 조합되었는지 추적합니다.

4. 암묵적 지식의 사멸 방지

최고 전문가의 직관적 판단 기준은 그 사람이 무대를 떠날 때 함께 소멸해왔습니다. 뇌 폴더는 개인의 암묵적 노하우를 디렉토리 계층구조로 포획하고, 상속 가능한 형태로 영구 보존합니다.

5. 다음 세대의 표준 선점

CLAUDE.md, AGENTS.md, .cursorrules — 세계는 이미 파일 시스템 기반의 AI 통제 방식으로 수렴하고 있습니다. 거시적인 행동 규칙을 제어하는 이 새로운 백본 규격을 구축하는 자가 다음 세대 AI 생태계를 장악합니다.

기능

왜 '폴더'가 벡터 검색을 압도하는가

📁

폴더 공간의 맥락화

단순한 OS 폴더 생성(mkdir)만으로 AI의 사고 체계가 논리적으로 구분되며, git diff를 통해 변화된 거버넌스 역사를 인간이 직관적으로 검수합니다.

⚡

빛의 속도로 읽어내는 트리

복잡한 RAG 파이프라인 없이 운영체제의 네이티브 Dentry 캐시를 활용하여 420개의 시냅스 트리 컴파일을 단 40ms 내외로 돌파합니다.

🔒

보안 및 에어갭 한계 돌파

외부 네트워크 송출은 물론 어떠한 감시망(Telemetry)도 내장되어 있지 않습니다. 극비 데이터 통제가 필수적인 온프레미스 망분리에 완벽히 부합합니다.

🎯

규칙의 자동 변용

하나의 통일된 지식 트리가 6개의 다른 개발 텍스트 레이어로 즉시 치환됩니다: GEMINI.md, .cursorrules, CLAUDE.md, AGENTS.md 등.

🧬

망각과 진화의 구현

내장된 TTL 감쇠 사이클과 헤비안(Hebbian) 학습 모터를 통해 사용되지 않는 규칙은 서서히 희석되고, 검증된 규칙은 핵심 공리로 강화됩니다.

💰

구조적 최소주의

무거운 종속성이나 구독 모델이 배제된 4MB의 모노리식 Go 바이너리. 리소스의 끝없는 비대화를 극도로 억제한 오픈소스(AGPL-3.0) 철학.

벤치마크

모든 벤치마크는 내부 테스트(go test ./...)입니다. 독립적 재현을 권장합니다.

활용 사례

엄격한 책임 소재가 요구되는 환경

글로벌 AI 책임법이 강화되는 시대, NeuronFS는 모든 결정과 규칙의 진화를 추적하여 감사 및 증명이 가능한 완벽한 거버넌스를 제공합니다.

⚖️

법무 및 컴플라이언스

"AI가 해당 결정을 내릴 때 어떤 사내 규정이 활성화되어 있었는가?" → git log brain/. EU AI Act를 대비한 결정적 감사 증명 수단입니다.

🏥

의료 시스템

명확히 추적 가능한 규칙 세트를 보유한 진단 보조 에이전트 구축. "AI가 이 환자를 왜 플래그 처리했는가?" → 버저닝된 금지(禁) 폴더가 책임의 근거를 입증합니다.

🏦

금융 및 금융심사

대출 승인 또는 거절 AI — 모든 가치판단이 특정 시점의 뇌 폴더 상태로 귀결됩니다. 컴플라이언스 담당자가 블랙박스 없이 로직의 버전을 검수합니다.

🎓

교육과 논리 검증

학생이 직접 규칙(禁/推 폴더)을 설계하고, AI가 그 룰 안에서 동작하며 논리적 허점을 역으로 드러내는 고차원적인 소크라테스식 교육이 가능합니다.

🚨

재난 대응 시스템

인터넷 인프라 붕괴 상태에서도, USB에 담긴 뇌 폴더와 로컬 모델의 결합으로 철저하게 통제된 지휘 구조(Chain of Command)를 유지합니다.

🔄

벤더 록인(Lock-in) 해방

GPT에서 Claude로 인프라를 이전해도 핵심 거버넌스는 그대로 유지됩니다. 더 이상 기업의 통제력이 API 벤더의 프롬프트 양식에 종속되지 않습니다.

비전

지식과 두뇌가 곧 상품이 됩니다

수천 번의 실패와 피드백을 통해 정제된 폴더 구조는 그 자체로 전송 가능한 '전문 지식 팩'입니다. 사람이 읽을 수 있고, 기계가 즉각 실행할 수 있는 역대 가장 완전한 가치관 생태계 시장.

💼

엔터프라이즈 두뇌 규격화

"시니어 프론트엔드 뇌" $29 · "금융 컴플라이언스 뇌" $199. 번거로운 튜닝 없이 단순 압축 해제만으로 전문가의 통제 기준을 시스템에 즉시 이식합니다.

🛒

AI 거버넌스 마켓플레이스

도메인, 업종, 직무별로 최적화된 거버넌스 마켓. 글로벌 커뮤니티가 치열하게 큐레이션한 베스트 프랙티스 정책들이 플러그인처럼 유통됩니다.

🧬

지식의 영구적 유산 보존

당신이 은퇴하더라도 의사 결정 구조는 남습니다. 30년 경력의 암묵적 룰이 뇌 트리로 캡슐화되어 영원이 실행됩니다. 인류 지식의 궁극적인 유언장.
