기능

왜 '폴더'가 벡터 검색을 압도하는가

NeuronFS는 파일시스템을 통해 AI 규칙을 구조화하고 계층화하는 통제 도구입니다. 복잡한 클라우드 없이 운영체제(OS) 네이티브의 속도와 결정성을 제공합니다.

📁

폴더 공간의 맥락화

OS 폴더 생성(mkdir)만으로 AI 통제 규칙이 성립됩니다. git diff를 통해 변화된 거버넌스 규칙과 회사의 방향성을 인간이 가장 쉽게 검수할 수 있습니다.

⚡

압도적으로 빠른 속도

수백 개의 통제 규칙을 뇌 폴더에서 읽어 들이는데 밀리초(ms)밖에 걸리지 않아 RAG의 레이턴시 지연을 완벽히 제거합니다. 개발자는 OS Dentry 메커니즘의 혜택을 누립니다.

🔒

최고 수준의 망분리

외부 네트워크 데이터 송출이 전무합니다. 어떠한 감시망(Telemetry)도 내장되어 있지 않아 엄격한 온프레미스 망분리에 완벽히 부합합니다.

🎯

규칙의 멀티 컴파일

단 하나의 지식 트리로 GEMINI.md, .cursorrules, CLAUDE.md 등 6가지 IDE 및 챗봇 규격에 맞게 동시 컴파일(치환)됩니다.

🧬

헤비안 진화 프로세스

자주 쓰인 규칙은 자동 강화되고, 사용되지 않은 규칙은 TTL 사이클을 거쳐 망각(Prune)되는 완전 자동 뇌 폐쇄 루프 통제기능이 적용되어 있습니다.

💰

범용성 & $0 평생 무료 구조

Mac, Windows, Linux 어디서나 완벽하게 동작합니다. 클라우드 구독이 필요 없는 단일 4MB 가벼운 실행 파일로 동작하는 오픈소스(AGPL-3.0)입니다.

명제

올바른 답보다 올바른 질문이 더 희귀합니다

1. 가치관의 최초 실행 가능한 포맷

인류는 수천 년 동안 가치관을 문자로 기록해왔지만, 스스로 실행될 수는 없었습니다. 뇌 폴더는 시대의 기치관이 인간의 통역 없이 직접 기계에게 실행되는 첫 번째 포맷입니다.

2. 통제와 책임의 인프라

"AI는 왜 그런 결정을 내렸는가?" 이제 시대는 이유를 묻습니다. 뇌 폴더와 Git 로그 체계는 감사 및 추적이 가능한 유일무이한 AI 거버넌스 프레임워크입니다.

3. 지식 유산 (Legacy)

최고 레벨 시니어의 경험적 직관은 그가 조직을 떠날 때 함께 소멸합니다. 뇌 폴더는 개인의 암묵적 노하우를 디렉토리 계층구조로 포획하고, 상속 가능한 형태로 영구 보존합니다.

아키텍처

7계층 포섭 (Subsumption) 통제

Rodney Brooks의 1986년 로봇 통제 구조(Subsumption Architecture)에서 착안했습니다. 하위 레이어의 본능이 시스템 최상위의 논리를 우선합니다 — 하드웨어 인터럽트와 동일한 원리입니다.

Brainstem (P0, 뇌간) 폴더에 禁 규칙이 담겨있다면, 상위 언어 모델이 제아무리 이를 거부하려 해도 NEVER-Tier로 절대 침범할 수 없이 강제 삽입됩니다.

객관주의 선언
이것은 프롬프트 생성의 통제를 보장하는 구조입니다. 즉, "출력의 확률"을 통제하는 것이 아닌, AI가 접하게 되는 "규칙 부여 단위"를 완전히 통제하는 것입니다.

P1Limbic스트레스, 패닉, 면역 반응, 경고 수준
P2Hippocampus장기 기억, 스킬 보존 영역
P3Sensors환경/운영체제 센서 등 환경 감지
P4Cortex일반 로직 (코드맵, 표준 규칙 등)
P5Ego톤앤매너, 화법, 프롬프트 성향
P6Prefrontal장기 전략 결정 및 마일스톤
작동 방식

단 세 번의 명령어. 의존성 제로.

01

규칙 생성 단계

규칙을 OS 상의 폴더 하나로 취급합니다. 깊이(Depth)가 구체성입니다.

mkdir brain/brainstem/禁/data_leak mkdir brain/cortex/dev/推/testing
02

컴파일 단계

트리를 순회하고, 계층 우선순위를 조정해 통합 마크다운으로 압축 조판합니다.

neuronfs --emit all ✓ 420 neurons → 6 files
03

통제 주입 (Deploy)

완성된 모델 파일이 즉각적으로 당신이 쓰는 IDE와 모델에 연동 구속됩니다.

Runewords

거버넌스 한자 옵코드

한자 표의문자와 특수기호 결합을 통해 하나의 디렉토리만 만들어도 AI가 행동 우선순위를 자동으로 분류합니다. (ALWAYS, WHEN→THEN, NEVER)

이것은 게임 디아블로 II의 룬워드(Runeword) 체계와 같습니다. 폴더가 소켓(Socket)이라면, 이 한자 접두어는 룬(Rune)입니다. 두 가지가 결합할 때 결정적 통제가 탄생합니다.

벤치마크

실제 테스트 데이터. 실제 숫자.

모든 벤치마크는 내부 테스트(go test ./...)에 근거합니다. 재현 가능 코드가 깃허브에 존재합니다.

공정한 선언 (Trade-Off)
난해하고 복잡한 질의(Hard Query)에서는 Vector DB의 압도적인 승리입니다. 하지만 엔터프라이즈 환경에서 "명시된 컴플라이언스 규칙"을 가져올 때 은유적 검색은 필요 없습니다. NeuronFS는 정확도를 일부 희생하는 대신, 서버 인프라 구축 제로와 기존 대비 20배 빠른 탐색 속도를 제공합니다.

활용 사례

완벽한 책임 소재가 요구되는 산업군

글로벌 AI 책임규정(EU AI Act 등)이 점점 강화되고 있습니다. NeuronFS는 모든 AI 규칙과 사고관의 진화 과정을 Git을 통해 추적 가능한 형태로 남깁니다.

⚖️

법무 및 컴플라이언스

"AI가 해당 결정을 내릴 때 어떤 사내 규정이 활성화되어 있었는가?" → git log brain/. 블랙박스였던 모델을 설명하는 것이 아니라, 애초에 어떤 규정이 담겼었는가를 법적 증거로 남깁니다.

🏥

의료 및 제약 시스템

명확히 추적 가능한 규칙 세트를 보유한 진단 보조 에이전트 구축. "AI가 어떤 근거로 이 환자를 주의 처리했는가?" → 버전 관리된 통제(禁) 폴더가 사후 책임의 근거를 입증합니다.

🏦

금융 및 심사 검증

대출 승인 또는 금융 청구 AI — 모든 가치판단이 특정 시점의 뇌 폴더 상태로 귀결됩니다. 금융 보안담당자가 텍스트 형태로 로직의 버전을 검수하고, 롤백할 수 있습니다.

🎓

논리 교육 시스템

학생이 직접 규칙(禁/推 폴더)을 조합하고, AI가 학생이 창조한 규율계 룰 안에서 동작하며 논리의 허점(Loophole)을 스스로 메우는, 쌍방향 소크라테스적 교육이 가능합니다.

🚨

재난 대응 오프라인 통제망

글로벌 인터넷망 붕괴나 랜섬웨어 사태 발생 시, USB에 담긴 이 가벼운 뇌 폴더와 로컬 거대 언어 모델이 결합하여, 가장 체계적이고 통제된 지휘 구조를 단 1문만에 부활시킵니다.

🔄

API 벤더 록-인(Lock-in) 해방

구글에서 만든 도구를 쓰다 오픈AI의 도구로 옮겨 탈 때마다 모든 환경을 새로 짜야 합니까? 이 뇌 구조를 갖추고 있다면, 당신 핵심 비즈니스 로직은 어떤 빅테크에도 종속되지 않습니다.

비전 마켓

당신의 노하우와 직관, 두뇌상품이 되다.

수천 번의 실패와 수정을 거쳐 완성된 완벽한 폴더 구조. 그것은 파일이 아니라 '추상적 직관'이 담긴 전문가의 두뇌 백업본입니다. 사람이 눈으로 읽고 기계가 직접 컴파일-제어하는, 지식 지적재산의 압축판(Zip)입니다.

💼

엔터프라이즈 맞춤형 두뇌

"10년차 B2B 카피라이팅 두뇌팩" $29. "스타트업 CTO의 검수 뇌" $199. 무거운 훈련 파이프라인 없이, 단순한 파일 압축 해제만으로 엘리트의 머릿속 기준이 팀에 삽입됩니다.

🛒

두뇌 팩 마켓 (Brain Marketplace)

도메인, 업종, 직무 단위로 파편화된 거버넌스 팩 생태계. 글로벌 AI 커뮤니티가 큐레이션한 최고 수준의 베스트 프랙티스들이 마치 플러그인처럼 교차 유통됩니다.

🧬

결코 사라지지 않는 유산 보존

전설적인 장인이 일선에서 물러나더라도, 그의 평생의 판단 기준과 철학은 남습니다. 그것은 뇌 트리로 캡슐화되어 사후에도 시스템을 통해 영구적으로 실행되며 남겨집니다.

모든 AI 파편을 단일 진실의 근원로 묶으십시오.

코어 정책을 한 번만 정의하면, 당신이 쓰는 모든 환경으로 동시 컴파일 됩니다. Git을 통해 투명하게 검증하세요. AI의 야생성을 확고한 통제망 안으로 가두는 단연코 가장 빠른 길입니다.
