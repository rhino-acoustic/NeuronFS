### 인메모리 마운트: 디스크에 0흔적

암호화된 `.brain` 파일 → 하드디스크에 안 풀고 **RAM에 가상 드라이브**로 마운트.

```
[일반 .zip]
풀기 → 하드디스크에 폴더 노출 → 누구나 복사 가능 😱

[암호화된 .brain]
엔진 실행 → RAM에만 마운트 → AI가 읽고 작동
→ 탐색기로 들여다보려 하면 → OS에서 접근 거부
→ 프로그램 종료 → RAM에서 증발 → 디스크 흔적 0
```

**"결과물은 누리되, 레시피는 훔칠 수 없는 완벽한 블랙박스."**

### Brainwallet: 세상 어디에도 키가 없다

> *"내 머리가 잊지 않는 문장"* → Argon2id → 마스터 키

```
"우리가 처음 만난 건 비 오는 화요일이었다"
        ↓ Argon2id (100ms 의도적 지연)
        ↓ SHA-256
AES-256 Master Key: 0x7f3a...
```

키를 USB에도, 클라우드에도, 종이에도 안 쓴다. **창조주의 기억 자체가 마스터 키.** 세상 어디에도 '저장'된 곳이 없으니 탈취 불가.

### DRM 시한폭탄

법과 규정은 매년 바뀐다. 암호화 키에 유효기간:

```
2026_개정세법_통제망.brain
→ 2026.12.31 24:00 → 복호화 키 파기 → 접근 불가
→ AI: "2027년형 라이선스를 갱신하십시오"
→ 고객: 매년 구독료 💰
```
