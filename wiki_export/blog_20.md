### 특허는 함정

| 특허 | 오픈소스 |
|---|---|
| 방어 범위 | 좁음 (폴더 구조 특허? 선행기술 풍부) |
| 비용 | 수천만 원 |
| 속도 | 2-3년 |
| 효과 | 대기업이 대충 변형해서 우회 |
| **시장 영향** | **없음** |

```
특허: "이건 내 기술이다" → 법적 싸움 → 스타트업 레벨에서 무의미
공개: "이건 새 카테고리다" → 커뮤니티 선점 → 표준이 됨
```

### "기술이 아니라 종교(Category)를 만들어라"

> *"기존에 없는 기술은 특허를 이길 수 없다. 대신 **카테고리를 만들어 종교의 창시자**가 돼라."*

- 리눅스는 특허를 안 냈다. 리눅스가 됐다.
- 워드프레스는 특허를 안 냈다. 워드프레스가 됐다.
- **NeuronFS는 특허를 내지 않는다. "폴더 기반 AI 거버넌스"라는 카테고리의 창시자가 된다.**

### 오픈코어 전략

```
[무료] 엔진 (Go 바이너리, MIT 라이선스)
  → 개발자 생태계 장악

  → 암호화된 전문가 지식

  → Brain Marketplace 수수료
```
